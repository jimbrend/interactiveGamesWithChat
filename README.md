# Chat Plays: A List of Chat Interactive Games

**A community-driven, ever-growing collection** of games where Twitch chat (and sometimes other platforms) directly influences, controls, or participates in the gameplay.

Whether your viewers are voting on disasters, racing marbles, commanding enemy armies, guessing drawings, or sabotaging your run — these games turn passive watchers into co-pilots (or chaotic gremlins). Perfect for increasing engagement, laughs, and community vibes on stream!

This README is designed to be **friendly, readable, and easy to contribute to**.

---

## Why These Games?

- Direct Twitch chat integration (votes, `!play`, `!pickme`, commands, polls, etc.)
- Chat affects **gameplay in real time** (not just overlays or chat bots)
- Works great for small or huge chats
- Includes free browser games **and** full Steam titles

> **Note:** Most games require a one-time Twitch login in the game settings. No extra bots needed for the core experience.

## Browser-Based / Free Chat Games
*(Zero downloads for viewers — just type in chat!)*

| Game                          | Price                  | How Twitch Chat Interacts                                                                 | Best For |
|-------------------------------|------------------------|-------------------------------------------------------------------------------------------|----------|
| **Marbles on Stream**         | Free                   | Viewers type `!play` (or `!play 1-6` for custom marbles) to join races. Chat votes on modes/maps. Supports up to **900 players**! | Huge chats, low-effort fun, world records |
| **Words on Stream**           | Free                   | Chat types guesses to solve scrambled anagrams/words. Real-time scoring and competition.   | Word games, quick Just Chatting segments |
| **Gartic on Stream** (Gartic Show) | Free              | Streamer draws → chat guesses by typing in Twitch chat (or vice versa). One-click Twitch connect. | Hilarious miscommunications & drawing chaos |
| **Kukoro: Stream Chat Games** | $3–$13 (Steam)         | 20+ mini-games. Viewers use chat commands to move, attack, vote, or battle the streamer.  | High-engagement variety |

## Games Enhanced by Tools (Popular Add-ons)

**Crowd Control** (Free service)  
Works with **150+ games**. Viewers spend bits/points to spawn items, enemies, buffs, or disasters in games like *Lethal Company*, *Content Warning*, *PlateUp!*, retro games, and more. Extremely popular for “let chat ruin my run” streams.

## How to Contribute (Super Easy!)

1. **Fork** this repository
2. **Edit** `README.md` (add new games in the same table format)
3. Add a short description of **how chat interacts** + price/platform
4. Submit a **Pull Request** with a clear title like `Added: New Game XYZ`

**Want to suggest a game without editing?** Open an Issue!

We also welcome:
- Screenshots or setup guides
- Updated links/prices
- New categories (e.g., “Mobile-friendly”, “2025+ releases”, “VR games”)
